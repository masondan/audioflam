# Nigerian Voice Samples for XTTS-v2

This folder contains reference audio samples for voice cloning with Replicate XTTS-v2.

## Requirements

- **Format:** WAV, MP3, M4A, OGG, or FLV
- **Duration:** 6-15 seconds of clear speech
- **Quality:** Clean audio, minimal background noise
- **Content:** Natural speaking voice (not singing, shouting, or whispering)

## Adding Voice Samples

1. Record or obtain a clear audio sample of a Nigerian English speaker
2. Name the file descriptively (e.g., `grace-female.wav`, `john-male.mp3`)
3. Place it in this folder
4. Update `src/lib/stores.ts` to add the voice to `REPLICATE_VOICES`

## Sample Sources

For public domain Nigerian voice samples, consider:
- Creative Commons licensed podcasts
- Public domain audiobooks by Nigerian authors
- Voice samples from open TTS datasets
